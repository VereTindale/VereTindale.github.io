Control flow and loops are ways of controlling a piece of code so that instead of writing it out a million times, you are able to build conditions inot the code that must be met, and of they are, then the code can repeat itself until it no longer meeets the conditions. An example would be if I am walking to my car and want to drive away, I cant run straight up to the car, throw the key in its general direction and assume I will drive away. If I did that i would run into the car and fall over and embarass myself (and get nowhere)...FIRST I unlock the door. I cant start the car until I get in the car. Second, I get in and close the car, etc etc. I'm kind of living in a loop of re-running the same thought of running through a checklist or requirements incremntally until I finally meet all requirtemd conditions and then STOP, because I have now begun driving my car.

The DOM is a live and interactive way of seeing your code in action that allows you to test things before commiting them etc.It canbe accessed in any browser (though there are diferences)

An Array can usually be thought of as a list of similar items or peices of information. An Object cna be a collection of different sets of data and can also hold different collections of arrays within it.


Functions are self contained peices of code which "do" something. They are lebelled and can then be called into other processes much faster by using their names, which define them and should be descriptive of their function within the code.
